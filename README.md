##Features

- Simple `Atomic<T>` class for numbers and strings.
- Uncomplicated `dispatch` keyword for firing off a new thread.
- Awesome `Chan<T>` for conncurent communication.
- Useful sync APIs. `Mutex`, `Cond`, `Once`, `WaitGroup`
